# Athletic Advantage
Source code for Athletic Advantage's landing page

## Config
[Main config](https://github.com/MrRyanSites/athletic-advantage/blob/main/src/lib/config.js) file is located in
```
./src/lib/config.js
```
## Development
Install dependencies
```bash
npm install
```
Start Development server
```bash
npm run dev
```
## Production Build
```bash
npm run build
```
